Tutorial/Hướng dẫn :
Hãy copy File đã giải nén vào file cài đặt game. MK: lequocdat8272.
Bạn không cần Replace hay ghi đè gì cả vì là mod.
Vào Game - config - Tiếng Việt - Khởi động lại Game.
